# 13623 Science in Context

## First Year

### 179 (12) Science in Context (2L, 1T)

This is a generic skills module in support of student success in Science. The module is divided into subunits
categorised as compulsory, strengthening and enrichment. Benchmark tests written at the start of the
academic year guide students as to which of the subunits they should take to make up the required 12
credits.
